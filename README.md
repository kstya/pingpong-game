🏓 Пинг-понг
Классическая игра в пинг-понг на Python и Pygame.


🎮 Описание
Игра для двух игроков с физикой мяча и подсчётом очков. Реализована с использованием объектно-ориентированного программирования.


🛠️ Технологии

Python 3
Pygame



🚀 Запуск
bashpip install pygame
python main.py
